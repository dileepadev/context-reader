# Project Documentation

This folder contains project-specific technical documentation for ContextReader.

## Recommended Reading Order

1. [PROJECT_PLAN.md](PROJECT_PLAN.md) to understand delivery phases and actionable tasks
2. [ARCHITECTURE.md](ARCHITECTURE.md) to understand design decisions and request/data flow
3. [DEPLOYMENT.md](DEPLOYMENT.md) to run the stack locally or prepare production deployment
